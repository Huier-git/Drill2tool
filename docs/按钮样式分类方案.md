# 按钮样式统一方案 - 按钮分类

## 1. Emergency (type="emergency") - 全局急停，红色加大
- ControlPage: `btn_stop_all` - 全部停止
- AutoTaskPage: `btn_emergency` - 急停
- SensorPage: `btn_stop_all` - 停止所有采集

## 2. Warning (type="warning") - 停止/断开/暂停，橙色
- VibrationPage: `btn_stop` - 停止采集, `btn_pause` - 暂停显示
- MdbPage: `btn_stop` - 停止采集
- MotorPage: `btn_stop` - 停止采集
- DrillControlPage: `btn_disconnect` - 断开连接
- ControlPage: 各个轴的停止按钮

## 3. Success (type="success") - 启动/连接/使能，绿色
- VibrationPage: `btn_start` - 开始采集
- MdbPage: `btn_start` - 开始采集
- MotorPage: `btn_start` - 开始采集
- SensorPage: `btn_start_all` - 启动所有采集
- DrillControlPage: `btn_connect` - 连接控制器
- ControlPage: `btn_enable` - 使能按钮
- AutoTaskPage: `btn_start` - 启动任务

## 4. Info (type="info") - 查询/日志/导出，浅蓝色
- DatabasePage: `btn_query` - 查询数据, `btn_export` - 导出数据
- 各页面的刷新/查询按钮

## 5. Primary (type="primary") - 主要操作，蓝色
- DrillControlPage: 运动控制按钮（上下升降等）
- ControlPage: 参数设置、移动按钮
- AutoTaskPage: 加载任务、保存任务等

## 6. Secondary (默认) - 次要操作，灰色
- 取消、关闭、返回等按钮
